ERROR CLASS 6: Answer Weakly Represented in Index / Insufficient Clue Vocabulary
==================================================================================

EXPLANATION:
A small set of questions fail not because the wrong entity type is retrieved, but
because the clue contains too little retrievable vocabulary, or the correct answer
article doesn't exist prominently enough in the Wikipedia subset being indexed. The
clue words match unrelated articles with equal or higher BM25 scores, leaving the
system with no good signal to work from. Fixes here require either a larger/more
current Wikipedia index or external knowledge sources (e.g., song databases for
'80s music, structured cultural references for niche topics).
