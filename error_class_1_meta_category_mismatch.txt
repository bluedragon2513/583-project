ERROR CLASS 1: Meta-Category Answer Type Mismatch
==================================================

EXPLANATION:
These categories embed a game rule inside their name (usually in a parenthetical
"Alex: ..." note) that redefines what the answer should be. The clue points to one
domain (a museum, a church) but the correct answer belongs to a different domain
entirely (a U.S. state, a capital city). BM25 retrieves documents from the domain
named in the clue, so the correct answer article is almost certainly absent from the
top-10 candidate list. LLM reranking cannot rescue a candidate that was never
retrieved. The fix requires a query rewrite that targets the *answer* domain — e.g.,
for STATE OF THE ART MUSEUM, searching Wikipedia's list of state articles for the
one that contains the museum name, rather than searching for the museum itself.
